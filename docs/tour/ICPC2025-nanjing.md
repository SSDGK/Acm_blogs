# icpc 50th南京站游记：
省流，打铁了，但是其实收获很多（物理意义上的）

经过一年的自学，终于在今天迎来了第一次icpc这样的超大型比赛。不得不提我和南京的确是有点缘，想起来之前暑假里看到洛谷上有一场2023南京的再现赛，我就拉上了@Zhangwuji 一起参加了一下，那一场似乎是我第一次vp区域赛？不太记得清了，但主要是这一场过题数我记得挺多，所以当时其实对南京的题好感度很高。原本我们计划是去哈尔滨的，因为一些特殊的原因得以改为前往南京，我其实挺高兴的，毕竟哈尔滨那么冷。

前天到了南京，南京毕竟是南方，虽然天一直阴着想要下雨的样子，但气温比较舒适，没有天津那么冷。我第一次入住亚朵这种偏高端的酒店还是非常激动的，当时还拍了个视频。

昨天去参加了开幕式，通过活动啥的拿到了很多礼品，真没想到居然还有滴蜡熊（虽然我想要初音fufu来着，但是扔沙包第一个力太大扔飞了/紧张）。给每个队伍送的袋鼠很可爱，符合南京特色。衣服挺好看的，就是号码选小了，我穿起来有点丑，估计要埋衣柜了。晚上还跑了8公里去出勤了，没错正式比赛前一晚上我还去了/高兴，主要是为了打卡。

今天正式赛，首先迅速过了C，然后就开始卡住了，其实K题一开始我就有想法，不过当时过的人少，我想不会这么简单吧……然后过了一会儿发现过了一堆，我就去猜了下，果然过了……然后就开始卡了，FGI到最后也没做出来，F想了非常多种做法，最后都还是否了，不过我最后想出来的做法是对的，只是需要一个剪枝优化，场上没想到。最后榜是3题就能铜牌，真的很遗憾。

打铁在我意料之中，从网络赛开始我就深知自己的能力远远不及铜牌，但今天离铜牌其实也就差个小优化，稍微有点不甘心，毕竟也已经尽力了，所以我也很快就能处理好情绪。我当初的一个小目标就是希望初战能够不打铁，现在没能达成说到底还是训练不足，不过我之前也想过这个事，我才入坑一年，但大部分其他选手都是高中甚至初中就开始准备了，我花的时间其实远远不及他们，因此我会败北也是情理之中。我对自己的表现没有不满意的，我已经尽了最大努力。

接下来还有一年时间，这一年我会做好计划，努力加练，明年卷土重来，一定拿下。
